## Quick start
1. Install dependencies:
```bash
npm install
```
2. Run dev server:
```bash
npm run dev
```
3. Build for production:
```bash
npm run build
```

## Deploy to GitHub Pages (recommended)
1. Create a repository on GitHub (e.g. `my-portfolio`).
2. Push this project to your repo.
3. In your GitHub repository settings → Pages, choose branch `gh-pages` or `main` and folder `/ (root)` or `docs` depending on your setup.
4. Or use `gh-pages` npm package to deploy the `dist` folder.
